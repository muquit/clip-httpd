## Release v1.0.1
This is the initial public release of clip-httpd, a simple, secure, and 
cross-platform clipboard server. This tool is designed to bridge the gap 
between your remote command-line environments and your local desktop, 
making it easy to copy text from any machine directly to your system's 
clipboard

* In Linux, the version that can resgisters on system tray, you might have to
do the following:

```bash
sudo apt update
sudo apt install libayatana-appindicator3-dev gir1.2-ayatanaappindicator3-0.1
```

Please look at [ChangeLog](ChangeLog.md) for what has changed in the current version.

Please do not forget to verify Checksums before installing and installing.
